# hermetic

Sync [Claude Code](https://docs.claude.com/en/docs/claude-code) plugins into
[Hermes](https://github.com/NousResearch/hermes-agent) as native skills and
delegation skills.

On PyPI as [`hermetic-cli`](https://pypi.org/project/hermetic-cli/); the CLI
itself is `hermetic`.

> 🚧 **Pre-alpha.** Documentation lands with v0.1 (see
> `docs/plans/2026-04-17-001-feat-hermes-plugin-sync-extraction-plan.md`).

## Status

Repository scaffold only. Full feature set, CLI reference, and quickstart
arrive once Unit 6 of the extraction plan lands.
